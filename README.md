## 🛠️ 技术栈
- **HTML5**：页面结构
- **CSS3**：响应式布局与动画效果
- **JavaScript (ES6)**：交互逻辑与本地存储
- **LocalStorage**：浏览器端数据持久化

## 📱 使用说明
### 基本操作
1. **打卡**：点击日历中的日期，然后使用"+"按钮增加打卡次数
2. **切换月份**：
   - 点击"←"、"→"按钮
   - 在日历区域左右滑动（移动端）
   - 点击"今日"按钮快速返回当前月
3. **查看统计**：
   - 本月打卡：当前月份的总打卡次数
   - 累计打卡：所有月份的总打卡次数
   - 连续打卡：最近连续打卡的天数
   - 本月连续未打卡：本月内最近连续未打卡的天数

### 数据管理
- 所有数据自动保存在您的浏览器中
- 更换设备或浏览器时数据不会同步
- 清除浏览器缓存会删除所有打卡记录

## 🌐 部署到 GitHub Pages
1. 进入仓库的 **Settings** 页面
2. 左侧菜单选择 **Pages**
3. 在 **Source** 部分选择 **main** 分支
4. 点击 **Save**
5. 等待几分钟后，访问 `https://[您的用户名].github.io/daily-check/`
